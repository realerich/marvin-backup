# 阿里云RDS PostgreSQL 配置指南

## 推荐配置

**产品：** 阿里云RDS PostgreSQL 基础版
**地域：** 华东1（杭州）或与你服务器同地域
**版本：** PostgreSQL 14或15
**规格：** 
- 1核2GB（起步，¥20-25/月）
- 或2核4GB（更流畅，¥50-60/月）
**存储：** 20GB SSD（起步，可自动扩容）

**预计费用：** ¥20-60/月

---

## 购买步骤

### 1. 进入购买页面
访问：https://www.aliyun.com/product/rds/postgresql

### 2. 选择配置
```
计费方式：包年包月（更便宜）或按量付费（灵活）
地域：与当前ECS服务器同地域（如华东1杭州）
数据库版本：PostgreSQL 14
系列：基础版（单机，性价比高）
CPU/内存：1核2GB
存储空间：20GB
```

### 3. 网络配置
```
网络类型：VPC（专有网络）
VPC：选择与你ECS相同的VPC
交换机：自动或选择同一可用区
```

### 4. 安全设置
```
数据库账号：marvin
密码：设置强密码（保存好！）
```

### 5. 完成购买
记录以下信息：
- 实例ID
- 内网地址（如：pgm-xxx.pg.rds.aliyuncs.com）
- 端口号（默认5432）
- 数据库账号和密码

---

## 配置白名单

购买完成后：

1. 进入RDS控制台 → 实例列表 → 管理
2. 左侧菜单：数据安全性 → 白名单设置
3. 添加当前OpenClaw服务器的公网IP到白名单
   - 如果ECS和RDS同VPC，使用内网地址，无需白名单
   - 如果是公网连接，需要添加ECS公网IP

---

## 连接信息示例

```
主机名：pgm-xxx.pg.rds.aliyuncs.com
端口：5432
数据库：marvin_db
用户名：marvin
密码：你的密码
```
